📦 ProjectJDBCtoJPA
📌 Descrição

Este projeto tem como objetivo demonstrar a evolução do acesso a dados em aplicações Java, realizando a transição de JDBC (Java Database Connectivity) para JPA (Java Persistence API).

A aplicação evidencia as diferenças entre uma abordagem mais manual (JDBC) e uma abordagem orientada a objetos utilizando ORM com JPA/Hibernate, que abstrai a manipulação direta de SQL e facilita a persistência de dados em bancos relacionais .

🎯 Objetivo
Comparar JDBC vs JPA na prática
Demonstrar boas práticas de persistência de dados
Facilitar o entendimento de ORM em Java
Servir como base para estudos com Spring Data JPA

🛠️ Tecnologias Utilizadas
☕ Java
🗄️ JDBC
🧩 JPA (Java Persistence API)
🔄 Hibernate (implementação ORM)
🐬 Banco de dados relacional
📦 Maven

⚙️ Funcionalidades
Conexão com banco de dados via JDBC
Execução de operações CRUD manualmente (SQL puro)
Mapeamento de entidades com JPA
Persistência de objetos utilizando ORM
Comparação entre:
Código verboso (JDBC)
Código simplificado e orientado a objetos (JPA)
